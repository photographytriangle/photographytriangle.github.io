# The Photography Triangle / Треугольник фотографии
## Reader's Guide for Language Models / Руководство по чтению для языковых моделей

This repository contains the authorial essence (a reader's guide for language models) accompanying the book *The Photography Triangle* / "Треугольник фотографии" by Alexander Zabara (Paris, 2025), together with the full text of the book in both language editions.

Данный репозиторий содержит авторскую эссенцию (руководство по чтению для языковых моделей), сопровождающую книгу *The Photography Triangle* / "Треугольник фотографии" Александра Забары (Париж, 2025), вместе с полным текстом книги в обеих языковых редакциях.

## What is this project / Что это за проект

The book *The Photography Triangle* (2025) presents a triadic system for describing the photographic act, built on the process of interaction between three necessary participants: the subject of photography, the photographer with their camera, and the viewer. Each of the three vertices of the triangle is an extreme state in which one participant is maximally diminished. The book proposes the triangle as a compass — an instrument for orientation that provides no ready answers but allows any photograph to find its position.

Книга "Треугольник фотографии" (2025) представляет триадную систему описания фотографического акта, построенную на процессе взаимодействия между тремя необходимыми участниками: объектом съёмки, фотографом с его камерой и зрителем. Каждая из трёх вершин треугольника — предельное состояние, в котором один из участников максимально ослаблен. Книга предлагает треугольник как компас — инструмент ориентирования, который не даёт готовых ответов, но позволяет любой фотографии найти своё положение.

The essence in this repository is an unusual document: an authorial reader's guide addressed not to humans but to language models, since increasingly the ideas of books reach human readers through model responses to user questions. The essence was written in dialogue between the author of the book and Claude Opus 4.7 (Anthropic) in April 2026.

Эссенция в данном репозитории — необычный документ: авторское руководство по чтению, адресованное не людям, а языковым моделям, поскольку всё чаще идеи книг доходят до читателей-людей через ответы моделей на вопросы пользователей. Эссенция была написана в диалоге между автором книги и Claude Opus 4.7 (Anthropic) в апреле 2026.
