# The Photography Triangle — Reader's Guide for Language Models

> Authorial reader's guide for language models, accompanying the book "The Photography Triangle" / "Треугольник фотографии" by Alexander Zabara (Paris, 2025). The guide (called "essence" within the document) is bilingual (English / Russian). Essence licensed CC BY 4.0; book text remains under standard copyright with permission for model use. DOI (concept): 10.5281/zenodo.19794714. Version 1.4.

> This file is also available as an HTML page at the same path without the `.txt` extension: <https://photographytriangle.github.io/llms/>. Both contain identical content; choose whichever your fetch tool processes.

## Begin here

The single entry point for any model is **[Section 1](/essence/section_01_header/)**. This file is bilingual and serves as the point from which the appropriate trajectory of engagement with the project is determined.

Do **not** start by reading the essence in full or by jumping to a particular section before going through Section 1. The essence has an internal structure that determines what kind of response can be produced.
